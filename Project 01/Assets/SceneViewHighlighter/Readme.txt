SceneView Highlighter

To change the settings, use "Edit->Preferences", goto "Highlighter" section.

Hover in Hierarchy: Highlight object when mouse hover the item in hierarchy.
  Modifier: Require a modifier key hold down to highlight object.
  
Hover in SceneView: Highlight object when mouse hover the item in scene view.
  Modifier: Require a modifier key hold down to highlight object.
  Show Name: Show the object name when mouse hover.
  Line to Pivot: Draw a line to the object pivot.
  
Outline Renderer: Enable outline renderer
  Color: Color of outline, fill using alpha value.
  Children Level: Max children level included to outline.
  Max Renderer: Max renderer to outline.
  
Outline Selected Sprites: Enable outline selected sprites
  Color: Color of outline, fill using alpha value.
  
Outline Buffer Half Size: Use half size buffer to draw.

Indicator: Draw indicator for non-renderer.
  Color: Color of indicator.
  
RectTransform Overlay: Draw overlay for RectTransform (UI elements)
  Color: Color of overlay, fill using alpha value.
